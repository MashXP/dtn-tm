Welcome to DTN Translation Manager

This is a user-friendly tool designed to simplify the process of managing and translating text within the DTN platform. With this manager, you can easily:

  • Edit and update translations
  • View translation history
  • Generate JSON files
  • Import JSON files
  • Download JSON files

This tool aims to streamline the translation process and ensure consistent and accurate translations entries.
